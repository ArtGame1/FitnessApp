# 🏋️‍♂️ FitnessApp — Мобильное приложение для фитнеса

> Учебный проект колледжа. Приложение для управления тренировками с тремя ролями: пользователь, тренер, администратор. Реализована авторизация через Firebase, динамические анимации упражнений, создание собственных тренировок и многое другое.

---

## 📱 О проекте

**FitnessApp** — это Android-приложение, разработанное в качестве учебного проекта. Приложение позволяет пользователям просматривать упражнения с анимациями, выполнять их, добавлять свои тренировки. Для тренеров и администраторов предусмотрены отдельные панели управления.

### Ключевые возможности
- 🔐 **Регистрация и вход** через Email или Телефон (Firebase Auth + Realtime Database)
- 👤 **Три роли**: Пользователь → Тренер → Администратор (с разными экранами)
- 🎬 **Анимированные упражнения** (AnimationDrawable для каждого упражнения)
- ➕ **Динамическое создание тренировок** (пользователь может добавлять свои упражнения)
- 🔍 **Поиск тренировок** (передача search_query между экранами)
- 📤 **Поделиться тренировкой** (Intent.ACTION_SEND)
- 📱 **Полноэкранный режим** (иммерсивный интерфейс без статус-бара)

## 🎭 Роли в системе

| Роль | Email | Пароль | Экран после входа |
|------|-------|--------|-------------------|
| **Администратор** | admin@fitnessapp.com | admin123 | AdminPanel |
| **Тренер** | coach@fitnessapp.com | coach123 | CoachActivity |
| **Пользователь** | Регистрация в приложении | (задаётся при регистрации) | MainActivity |

> 💡 **Секретная функция**: Долгое нажатие на логотип на экране входа показывает подсказку с данными для входа админа и тренера.

## 🚀 Основной функционал (по экранам)

### 1. LoginActivity — экран входа
- **Динамическое поле ввода**: Переключение между Email и Телефоном (меняет hint и тип клавиатуры)
- **Валидация пароля**: длина ≥ 6 символов, обязательно наличие цифры
- **Локальные учётные записи**: admin@fitnessapp.com / coach@fitnessapp.com (вход без интернета)
- **Вход по телефону**: Поиск email в Realtime Database → стандартный Firebase-вход
- **Полноэкранный режим**: скрыты статус-бар и навигационная панель

### 2. WorkoutActivity — список тренировок
- **10 встроенных упражнений** с анимированными иконками (AnimationDrawable)
- **PopupMenu** с действиями:
  - ✏️ Редактирование (заглушка)
  - ➕ Добавить свою тренировку (диалог с названием и описанием)
  - 📤 Поделиться (ACTION_SEND)
  - 🔄 Обновить (заглушка)
  - 🗑️ Удалить (подтверждение → finish())
- **Динамические карточки**: создаются CardView + TextView + ImageView с анимацией
- **Переход к упражнению**: передача day, exerciseName, exerciseImageResource в ExerciseActivity
- **Поиск**: получает search_query из Intent (фильтрация пока не реализована)

### 3. ExerciseActivity (упоминается в коде)
- Получает параметры из Intent и отображает конкретное упражнение
